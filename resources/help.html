Keyboard shortcuts

  • [Space] - Play/Pause
  • [F] / [B] - Word forward / backward
  • [A] - Play again
  • [O] - Open text file
  • [Q] - Quit program
  • [Ctrl+V] - Read text from keyboard
  • [Alt+F] - Adjust font
  • [S] - Open settings dialog
  • [Up] / [Down] - Increase or decrease speed
